SNApplication

new SNApplication(environment, platform, namespace, deviceInterface, swapClasses, skipClasses, crypto)

Source:
  • application.js, line 53
Parameters:
Name Type Description
environment Environments Required - The Environment that identifies your application.
platform Platforms Required - The Platform that identifies your application.
namespace string Optional - a unique identifier to namespace storage and other persistent properties. Defaults to empty string.
deviceInterface DeviceInterface A DeviceInterface object.
swapClasses array Gives consumers the ability to provide their own custom subclass for a service. swapClasses should be an array of key/value pairs consisting of keys 'swap' and 'with'. 'swap' is the base class you wish to replace, and 'with' is the custom subclass to use.
skipClasses array An optional array of classes to skip making services for.
crypto SNCrypto The platform-dependent instance of SNCrypto to use. Web uses SNWebCrypto, mobile uses SNReactNativeCrypto.
